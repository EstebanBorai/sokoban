specs.

# Usage

```bash
git clone https://github.com/EstebanBorai/sokoban.git

cd ./sokoban

cargo run
```

# Controls

Key | Description
--- | ---
<kbd>W</kbd> or <kbd>Up</kbd> | Move north
<kbd>A</kbd> or <kbd>Left</kbd> | Move west
<kbd>S</kbd> or <kbd>Down</kbd> | Move south
<kbd>D</kbd> or <kbd>Right</kbd> | Move east
